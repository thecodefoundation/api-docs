Getting Started

Aify is very easy to get started with. Following this guide you can begin using Aify in under 15 minutes! Let’s dive in!

Step 1 - Create your account on Aify Dashboard

  1. Head over to aify.thecodefoundation.dev. You’ll be presented with the login screen as shown below -
  2. Click on Create an Account to register a new account. If you already have an account, skip to step 4.
  3. Fill in the required details on the registration page. Click on Register to button to complete registration.
  4. You’ll be taken back to the login screen. Enter your credentials and submit the form to login.

Step 2 - Add your website

  1. On the left menu, Click on Websites > Add
  2. Add your website, enter any name for your website and enter the root URL of the site.

Ensure that your website has a publicly accessible sitemap file, preferably at [/sitemap.xml](https://example.com/sitemap.xml) location. Our Aify Crawler will use this sitemap to crawl your website.

Wait for a few minutes for the Aify Crawler to visit your website and work its magic in the background. Meanwhile, you can move to the next steps and grab your API keys!

Step 3 - Get your API Keys

  1. On the top right of any page in the Aify Dashboard, Click on the user profile picture to open your personal information and settings menu.
  2. Click on Settings to open up your account settings page.
  3. To create a New API Key, enter any string as per your wish as a name for the Key. Click on Create API Key button.
  4. Note down the API key displayed in a safe location. You will not be shown in this API key again.

Step 4 - Play with the APIs

Now that you’ve got your API keys, you can play around with your API key using Postman or any other equivalent tool for making calls to the REST API.
